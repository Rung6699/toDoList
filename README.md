# todo-list

### Step to run

1. Clone this project

```
git clone  https://github.com/RTae/vue-todo-list.git
```
2. Change directory folder to the project folder

```
cd vue-todo-list
```

3. Install package

```
yarn install 
```

4. Run project

```
yarn serve 
```

ถ้ารันเเล้วกด Edit เเล้วข้อมูลยังไม่เปลี่ยนเเปลง ให้ลงกด Refresh ดูนะครับ
